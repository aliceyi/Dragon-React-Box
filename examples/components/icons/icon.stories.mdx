# Icon

Default icons in the system.

# Install

```
    yarn add @ff-dragon/react-icons
```

# Import

```
    import { Icons } from @ff-dragon/react-icons
Or
import Icons from @ff-dragon/react-icons
```

# Use

```
    import { Icons } from @ff-dragon/react-icons
<Icons type='IconEdit' {...args} />
Or
import IconItem from @ff-dragon/react-icons
<IconItem.IconEdit type='IconEdit' {...args} />
```
